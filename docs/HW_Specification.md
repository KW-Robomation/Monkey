# Plutto Hardware Specification

## 1. 개요

본 문서는 로봇팔 시스템 **Plutto**의 하드웨어 구성과  
소프트웨어 제어를 위해 정의된 **기구학적 기준 및 제원**을 설명합니다.

해당 문서는 관절 구조, 좌표계 정의, 각도 원점, 가동 범위 등  
소프트웨어에서 로봇 동작을 생성하고 해석하기 위해 필요한  
**하드웨어 기준 정보(reference)** 를 제공하는 것을 목적으로 합니다.

본 문서에 정의된 기준은 시뮬레이터(Sketch), 동작 로직(Spine),  
향후 실제 하드웨어 제어 모듈에서 **공통 참조 기준**으로 사용됩니다.

---

## 2. 로봇 하드웨어 구성 개요

### 2.1. 로봇 타입

![alt text](https://github.com/user-attachments/assets/7d4f019c-7e48-47ed-a484-2385316aa2e9)

- 로봇 형태: **2-DOF Planar Drawing Robot Arm**
- 구동 방식: 회전 관절 기반 (Rotational Joint)
- 말단 장치(End Effector): 펜 홀더 (Pen Holder)

### 2.2. 관절 구성

|  관절   | 명칭        | 설명                                         |
| :-----: | :---------- | :------------------------------------------- |
| Joint 1 | Base Joint  | 로봇 몸체에 고정된 회전 관절, 좌우 회전 담당 |
| Joint 2 | Elbow Joint | 말단부 이동을 담당하는 두 번째 회전 관절     |

---

## 3. 좌표계 정의

### 3.1. 기준 좌표계 (World Coordinate)

- 원점 (0, 0): 로봇 베이스 중심
- X축: 로봇 기준 우측 방향
- Y축: 로봇 기준 전방(또는 상단) 방향

> SVG 좌표계는 별도의 변환 과정을 통해 본 좌표계로 매핑된다.

### 3.2. 로봇 좌표계

- 모든 기구학 계산은 **로봇 기준 좌표계**를 사용한다.
- 시뮬레이션 및 하드웨어 제어는 동일한 좌표계를 공유한다.

---

## 4. 관절 각도 정의

### 4.1. 관절 각도 기준 (Zero Position)

|  관절   | 각도 원점 정의                               |
| :-----: | :------------------------------------------- |
| Joint 1 | 로봇 팔이 정면을 향할 때를 0°로 정의         |
| Joint 2 | 링크가 완전히 펴진 상태를 기준으로 0°로 정의 |

- 모든 관절 각도는 **시계 반대 방향(CCW)** 을 양(+)의 방향으로 정의한다.
- 각도 단위는 **degree(°)** 를 기준으로 한다.

### 4.2. 관절 회전 방향

- Joint 1: CCW → 좌측 회전
- Joint 2: CCW → 말단부 상승 방향

## 7. 펜 메커니즘

### 7.1. 펜 상태 정의

|   상태   | 설명                                    |
| :------: | :-------------------------------------- |
|  Pen Up  | 펜이 종이에서 떨어진 상태 (이동만 수행) |
| Pen Down | 펜이 종이에 접촉한 상태 (드로잉 수행)   |

### 7.2. 소프트웨어 제어 기준

- 펜 상태는 관절 동작과 독립적인 제어 명령으로 처리된다.
- Plutto Path 포맷에서는 별도의 Control Byte로 표현된다.

---

## 8. 소프트웨어 연동 기준

본 문서에 정의된 하드웨어 제원은 다음 모듈에서 참조된다.

- **Spine**: 기구학 계산, 관절 각도 검증
- **Sketch**: 로봇 위치 및 드로잉 시각화
- **Dashboard**: 사용자 입력 제한 및 실시간 제어

향후 실제 하드웨어 제어 모듈이 추가될 경우,  
본 문서의 기준을 그대로 사용하여 제어 일관성을 유지한다.

---

## 9. 변경 및 확장 고려 사항

- 관절 수 증가 시 (예: Z축, 손목 관절)
- 펜 외 도구 장착 시 (브러시, 마커 등)
- 속도, 가속도, 토크 제한 파라미터 추가 시

본 문서는 하드웨어 확장에 따라 지속적으로 갱신될 수 있다.
